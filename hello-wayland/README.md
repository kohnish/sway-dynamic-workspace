# hello-wayland

A hello world Wayland client, 2018 edition.

This is a simple client showing a picture. It uses the [xdg-shell] protocol.

## Dependencies

The following dependencies are required for the Makefile to function properly:

- libwayland
- wayland-protocols
- ImageMagick
